# Basic-HTML-Website
Taste Buds Restaurant Website
📌 Project Description

Taste Buds is a simple and responsive restaurant website created using HTML and CSS.
The website provides visitors with information about the restaurant, its menu, customer reviews, and contact details. It is designed to give users a quick overview of the restaurant and encourage them to visit or place an order.
Features-
Home Page – Introduction to the restaurant
About Us Page – Information about the restaurant and its story
Menu Section – Displays available dishes including vegetarian thali
Customer Reviews – Shows feedback from customers
Contact Us Page – Contact details and location information
Scrolling Text Feature – Highlight messages like Best Rated Restaurant
Simple Layout – Easy navigation between pages
